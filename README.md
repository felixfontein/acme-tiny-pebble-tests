# acme-tiny-pebble-tests

Basic tests for [acme-tiny](https://github.com/diafygi/acme-tiny/) using the [Pebble](https://github.com/letsencrypt/Pebble) ACME CA.

## Usage

Building the image locally
```
./build-local-master.sh
```

Fetching `acme_tiny`
```
./get-acme-tiny.sh
```

Running the tests
```
./run.sh
```
